Operating System From Scratch
-----------------------------

Step 04: Boot loader
````````````````````

Let's write a boot loader.
The booting process will be:

+ power on, BIOS loads boot sector
+ boot sector loads boot loader
+ boot loader loads kernel
+ kernel runs
